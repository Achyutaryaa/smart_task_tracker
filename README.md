# 🧠 Smart Task Tracker

A simple and efficient task management web app that helps you **plan, track, and stay productive**.
This project stores tasks in the browser using **Local Storage**, so your data persists even after refreshing the page.

---

## 🚀 Features

* ➕ Add tasks with categories (Work, Personal, Study)
* 🗂️ Filter tasks by category
* 💾 Persistent storage using browser Local Storage
* 🎨 Clean and responsive UI
* ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Logic and functionality
* **Local Storage API** – Data persistence

## ⚙️ How It Works

* Tasks are stored using the browser's **Local Storage**
* When you add a task:

  * It is saved in Local Storage
  * It is rendered on the UI
* On page reload:

  * Tasks are fetched from Local Storage
  * Displayed automatically

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/smart_task_tracker.git
   ```

2. Navigate to the project folder:

   ```bash
   cd smart_task_tracker
   ```

3. Open `index.html` in your browser
